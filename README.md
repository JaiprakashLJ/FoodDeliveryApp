# 🍕 FoodDeliveryApp — Full Stack Food Delivery Application

A Swiggy/Zomato-style food delivery web application built with Java EE.

## Tech Stack
- **Backend** — Java Servlets, JDBC, JSP
- **Frontend** — HTML, CSS
- **Database** — MySQL
- **Server** — Apache Tomcat
- **Security** — BCrypt password hashing

## Features
- 🏠 Restaurant listing page
- 🍽 Dynamic menu per restaurant
- 🛒 Session-based cart system
- 👤 User login & signup with BCrypt
- 📦 Order placement & history
- 🛡 Admin role support

## How to Run
1. Import project into Eclipse
2. Setup MySQL database — run SQL scripts
3. Add MySQL & jBCrypt JARs to WEB-INF/lib
4. Deploy on Apache Tomcat
5. Visit `http://localhost:8080/FoodApp/`
